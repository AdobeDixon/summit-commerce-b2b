# Order New Delivery Block

## Overview

The Order New Delivery block is a multi-step wizard for placing Bodea delivery orders. Steps: Order Type, Delivery Date, Transport, Equipment, Site & Contact, Delivery Window. Uses Commerce order API and equipment product catalog.

## DA.live Integration

- **Rows**: 1
- **Columns**: 1
- Single cell; all UI is generated by the block

## Configuration

No section metadata. Configuration lives in `sites.js`, `equipment-products.js`, `commerce-config.js`.

## Behavior

- Step-by-step wizard with validation
- Calendar date picker, equipment capacity cards, site search
- Submits via `submitOrderWizard` in `submission.js`

## Security

- Uses `escapeHtml()` for user-entered values in templates
- No unsanitized author content in innerHTML

## Accessibility

- Semantic form controls
- Ensure `:focus-visible` on buttons and inputs
- Respect `prefers-reduced-motion` for calendar/animations
